# Demo Assets

Place portfolio media files in this directory.

Recommended files:

- `demo-flow.gif` - short full flow (init -> ask -> snapshots -> update-pack).
- `ui-dashboard.png` - Streamlit dashboard screenshot.
- `api-docs.png` - FastAPI Swagger docs screenshot.
- `outputs-sample.png` - outputs folder snapshot with generated artifacts.

Tip:

- Keep image width around 1400-1800px for clear GitHub rendering.
- Keep GIF under 10-15 MB for fast loading.
